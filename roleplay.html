Website Roleplay

Sistem komunitas roleplay modern berbasis web

Deskripsi Proyek

Website ini dirancang untuk komunitas Roleplay dengan fitur lengkap seperti login, forum, server list, dan dashboard pengguna. Sistem ini dibuat agar komunitas bisa berinteraksi, berdiskusi, dan mengelola data roleplay dengan mudah.

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js (Express.js)
  • Database: MongoDB
  • Real-time Chat: Socket.io

Fitur Utama

  • Login & Registrasi pengguna
  • Forum & sistem komentar
  • Manajemen server komunitas
  • Role & permission seperti Discord
  • Chat real-time menggunakan Socket.io
