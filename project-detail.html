Script Reconnect SA-MP

Deskripsi Proyek

Script ini dibuat menggunakan Lua + MonetLoader untuk pemain SA-MP di Android. Fungsinya agar bisa melakukan reconnect otomatis tanpa harus keluar game dengan /quit. Fitur ini berguna ketika server lag atau tidak ada suara tanpa perlu restart client.
